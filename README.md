# BIS Standards Recommendation Engine

AI-powered RAG system that maps product descriptions to relevant Bureau of Indian Standards (BIS) for Building Materials.

Built for the **BIS x Sigma Squad AI Hackathon** — Track: RAG / AI.

## Setup

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Get a free API key

**Groq** — for LLM inference only:
1. Go to https://console.groq.com/keys
2. Click "Create API Key" — free tier, very fast

### 3. Set API key
Edit `.env`:
```
GROQ_API_KEY=your_groq_key
```

### 4. Build the index (run once)
```bash
python src/ingest.py
```
This parses `dataset.pdf`, chunks by BIS standard, embeds with `all-MiniLM-L6-v2` (local, no API needed), and saves `index/chunks.pkl`.
Takes ~30 seconds on CPU.

## Architecture

```
Product Description
       │
       ▼
all-MiniLM-L6-v2 (local, sentence-transformers)
       │  (query embedding, no API needed)
       ▼
Cosine Similarity Search
over index/chunks.pkl  (559 BIS standards)
       │  (top-5 BIS standard chunks)
       ▼
Groq llama-3.3-70b-versatile
       │  (rationale generation)
       ▼
Validated Results
(standard_id + rationale)
```

**Chunking strategy:** One chunk per BIS standard summary. Each chunk contains the full text of one IS standard from SP 21 (2005). 559 standards indexed. This ensures exact IS number retrieval.

**Embeddings:** `all-MiniLM-L6-v2` via `sentence-transformers` — runs fully locally on CPU, no API key or internet required at inference time.

**Anti-hallucination:** LLM output is validated — any standard ID not present in the retrieved set is discarded before returning results.
